Счастливые числа - это те числа, которые в результате ряда преобразований вида "сумма квадратов входящих цифр" превратятся в единицу.

Например:

7 => 7^2 = 49, 49 => 4^2 + 9^2 = 16 + 81 = 97, 97 => 9^2 + 7^2 = 81 + 49 = 130, 130 => 1^2 + 3^2 + 0^2 = 10, 10 => 1^2 + 0^2 = 1.

Следовательно число 7 - счастливое.

Введите число и проверьте, счастливое ли оно.
